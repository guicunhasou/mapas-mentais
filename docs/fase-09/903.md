# 903
## Introdução ao Adobe After Effects

- Configuração de Hardware
  - Requisitos Mínimos
    - Processador Quad-core
    - 16 GB RAM
    - 2 GB VRAM GPU
  - Requisitos Recomendados
    - 8 núcleos ou mais
    - 32 GB RAM ou mais
    - SSD de alta velocidade

- Interface do Software
  - Painel de Projeto (Arquivos)
  - Painel de Composição (Visualização)
  - Painel de Propriedades (Atributos)
  - Linha do Tempo (Timeline)
  - Menu Suspenso (Ferramentas)

- Otimização de Desempenho
  - Interpolação espacial linear
  - Centralizar ponto de ancoragem
  - Configuração de Cache de disco
  - Salvamento automático (5 min)
  - Personalização de Rótulos

- Recursos de Animação
  - Keyframes (Quadros-chave)
  - Interpolação de movimento
  - Propriedades de Transformação
    - Posição (X, Y)
    - Escala (Tamanho)
    - Rotação (Ângulo)
    - Opacidade (Transparência)

- Atalhos Essenciais
  - Ctrl+N: Nova Composição
  - Ctrl+D: Duplicar Camada
  - Barra de Espaço: Mover Timeline
  - F2: Desmarcar Tudo
  - Alt+Shift+S: Criar Keyframe
