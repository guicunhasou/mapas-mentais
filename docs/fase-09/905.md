# 905
## Passes de Mágica: Transições e Animações

- Técnicas de Transição
  - Importância
    - Guia o usuário
    - Feedback imediato
    - Disfarça carregamento
    - Identidade da marca
    - Acessibilidade cognitiva
  - Shape Layer (Elipse)
    - Ferramenta Forma
    - Reposicionar Ponto Âncora
    - Animação de Rotação
    - Easy Ease (F9)
    - Efeito Preenchimento
  - Transição de Linhas
    - Ferramenta Caneta
    - Efeito Zigue-Zague
    - Arremate Redondo
    - Aparar Caminhos (Trim Paths)
    - Influência de Interpolação (70%)
    - Pré-composição

- Navegação e Fluxo
  - Animação de Botão
    - Sólido (BG)
    - Retângulo Arredondado
    - Ferramenta Texto (PLAY)
    - Animação de Escala
    - Parenteamento (Parent Link)
  - Feedback Visual
    - Efeito de Onda (Traçado)
    - Editor Gráfico de Velocidade
    - Camada de Clique
    - Máscara Geral (Track Matte)

- Princípios Básicos
  - Bouncing Ball (Bola Quicando)
    - Configuração de Composição
    - Anchor Point na base
    - Dimensões Separadas (X e Y)
    - Gráfico de Valores
    - Deformação por Escala (Realismo)

- Conclusão
  - Interfaces intuitivas
  - Movimentos fluidos
  - Design profissional
  - Redução de fricção
