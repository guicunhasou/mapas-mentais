# 909
## Memória de Elefante: Persistência e Recursos Nativos

- AsyncStorage
  - O que é
    - Armazenamento local chave-valor
    - Operações assíncronas
    - Persistência entre sessões

  - Operações Básicas
    - setItem (salvar)
    - getItem (recuperar)
    - removeItem (remover)

  - Boas Práticas
    - Evitar dados sensíveis
    - Compressão de dados grandes
    - Limpeza de dados obsoletos

  - Casos de Uso
    - Configurações de app
    - Preferências de tema
    - Estado de login
    - Cache simples

- Firebase Firestore
  - Banco NoSQL na nuvem
  - Sincronização em tempo real
  - Suporte a modo offline
  - Escalabilidade para múltiplos dispositivos

- Recursos Nativos
  - Câmera
    - react-native-vision-camera
    - expo-camera
    - Captura de fotos e vídeos

  - GPS (Geolocalização)
    - react-native-geolocation-service
    - Coordenadas em tempo real
    - Permissões Android/iOS

  - Sensores
    - Acelerômetro
    - Giroscópio
    - Magnetômetro

  - Funcionamento
    - Bridge (Ponte) entre JS e Nativo
    - Módulos pré-construídos

- Projeto Prático: Scanner QR
  - Tecnologias
    - Vision Camera (Scanner)
    - AsyncStorage (Storage offline)

  - Configuração Android
    - AndroidManifest (Permissões)
    - gradle.properties (Build)

  - Configuração iOS
    - Info.plist (Mensagens de uso)

- Critérios de Escolha
  - Popularidade da comunidade
  - Compatibilidade com versão RN
  - Qualidade da documentação
  - Necessidade real de dependência
