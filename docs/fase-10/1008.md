# 1008
## Publicação de Aplicativos Híbridos

- Google Play (Android)
  - Requisitos Iniciais
    - Google Play Console
    - Taxa única de inscrição
    - Certificados de assinatura (Keytool)
  - Formatos de Build
    - APK (Tradicional)
    - AAB (Recomendado/Otimizado)
  - Processo de Submissão
    - Criar novo projeto
    - Inserir metadados e recursos
    - Upload do arquivo AAB
    - Configurar distribuição e preços
    - Enviar para revisão
- App Store (iOS)
  - Requisitos Iniciais
    - Apple Developer Program
    - Taxa anual
    - Certificados e Perfis de Provisão
  - Gerenciamento
    - App Store Connect
    - Xcode (Geração de IPA)
    - TestFlight (Testes)
  - Submissão
    - Archive no Xcode
    - Upload via Xcode Organizer
    - Configurar metadados no portal
    - Agendamento de lançamento
- Automação com Fastlane
  - Ferramenta Open-source
  - Instalação via Ruby
  - Configuração de Lanes (Fastfile)
  - Automatização de build e upload
- Boas Práticas
  - Testes Extensos
    - Funcionais
    - Compatibilidade
    - Performance
    - UI e Automatizados
  - Documentação
    - Política de Privacidade
    - Termos de Uso
    - Notas de Versão
  - Otimização
    - ProGuard (Minimização)
    - Lazy Loading
    - Monitoramento (Crashlytics/Sentry)
  - Feedback
    - Monitoramento de avaliações
    - Resposta ativa
    - Coleta in-app
